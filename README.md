## Brief description of the site
This site is a collection of recipes that my mom has in her recipe book. First there is a homepage with featured recipes from different categories as well as space to sign up for emails, although it isn't working yet. Next there is a page of 15 recipes from my mom's cookbook with toggles to different types of meals and preferences. The user is able to click on each recipe card and the ingredients and steps will show in a pop up page. The last page is an about me page about the website and contact information.

## What you learned
I learned that it is important to ask your client for specific details for what they like about a website and what they don't because having a good understanding of the specifics they are looking for makes it easier to create a clear website that they would like. I also learned that creating a sketch image is extremely helpful and the more detailed the image is, the easier it will be to bring that to the computer and make the website real. Something else I learned is that it is difficult to even create a simple website. I wanted to include AI features and a subscription system but even getting all of the pages to work was difficult. A main thing I learned was the recipes.json I created. I learned that syntax is extremely important and even one syntax error can cause the entire website to crash. I spent awhile getting the json correct so the recipe cards would appear and when I would be able to do that then there would be no recipe, so it was hard getting all of those features to cooperate.
